# HES ICD-10 Code Lists for Long-Term Conditions

## Overview

This repository contains the ICD-10 code lists used to identify long-term conditions (LTCs) and fracture outcomes in Hospital Episode Statistics (HES) for my study. 

The code lists are provided to support transparency, reproducibility, and reuse of the research methods.

## Repository Contents

This repository includes ICD-10 code lists for the following conditions and outcomes:

- Type 1 Diabetes
- Type 2 Diabetes
- Sleep Apnoea
- Asthma
- Cancer
- Chronic Kidney Disease (CKD)
- Chronic Obstructive Pulmonary Disease (COPD)
- Dementia
- Depression
- Hip Fracture
- Hypertension
- Liver Disease
- Nephropathy
- Neuropathy
- Osteoarthritis
- Osteoporosis
- Pelvic Fracture
- Proximal Humerus Fracture
- Radius Fracture
- Vertebral Fracture
- Rheumatoid Arthritis
- Stroke

Each worksheet contains the ICD-10 codes and corresponding diagnostic terms used to define these conditions within HES.

## Data Source

The code lists were developed using the International Classification of Diseases, 10th Revision (ICD-10), and were used to identify diagnoses recorded in Hospital Episode Statistics (HES).

Researchers should ensure that these code lists are appropriate for their study population and coding practices.
